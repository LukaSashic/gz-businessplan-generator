## Overview

The GZ Businessplan Generator supports 15 distinct business types. Each type has specific:
- Question variations per module

- Financial templates and patterns
- Coaching focus areas
- Common limiting beliefs

## 1. Beratung (Consulting)

### Profile
- **Category:** Service
- **Target:** B2B
- **Pricing:** Hourly (€80-200) or Project-based
- **Capital Needs:** Low (€5,000-15,000)
- **Location:** Home office / Remote

### Module Variations

**Geschäftsmodell:**
- Focus on specialization vs. generalist positioning
- Emphasize expertise proof (certifications, track record)

**Finanzplanung:**
- Utilization rate: realistic max 60-70%
- Consider acquisition time between projects
- Tagessatz calculation: monthly needs ÷ billable days

### Validation Rules
- Max utilization: 70%
- Min Tagessatz for sustainability: €800
- Realistic first-year clients: 3-8

### Coaching Focus
- Limiting belief: "Der Markt ist gesättigt"
- Challenge: Niche differentiation
- Common anxiety: "Woher kommen Kunden?"

### Financial Template
```
Revenue Model: Tagessatz × Billable Days
Typical Tagessatz: €800-1,500
Billable Days/Month: 12-15
Annual Revenue Y1: €100,000-180,000
```

## 2. Agentur (Agency)

### Profile
- **Category:** Service
- **Target:** B2B
- **Pricing:** Project-based, Retainer
- **Capital Needs:** Medium (€20,000-50,000)
- **Location:** Office recommended

### Module Variations

**Unternehmen:**
- Team structure critical
- Roles: Account, Creative, Tech

**Finanzplanung:**
- Higher fixed costs (office, team)
- Consider freelancer network costs
- Project profitability varies

### Validation Rules
- Min team for agency label: 2+ people
- Office costs realistic for location
- Project margins: 30-50%

### Coaching Focus
- Scaling vs. boutique decision
- Team management challenges
- Cash flow with project delays

### Financial Template
```
Revenue Model: Project Fees + Retainers
Avg Project: €5,000-50,000
Retainer: €2,000-10,000/month
Team Cost: 40-60% of revenue
```

## 3. Freiberufler (Freelancer)

### Profile
- **Category:** Service
- **Target:** B2B + B2C
- **Pricing:** Hourly
- **Capital Needs:** Very Low (€2,000-8,000)
- **Location:** Home office

### Module Variations

**Intake:**
- Check Freiberufler status (§18 EStG)
- No Gewerbe needed for: writers, artists, teachers, engineers, consultants

**Finanzplanung:**
- No Gewerbesteuer
- Lower social contributions possible
- Künstlersozialkasse eligibility

### Validation Rules
- Confirm Freiberufler eligibility
- Realistic hourly rate for profession
- Max working hours: 40-50/week

### Coaching Focus
- Work-life balance
- Rate negotiation confidence
- "Ich bin kein Verkäufer" belief

### Financial Template
```
Revenue Model: Stundensatz × Hours
Typical Rate: €50-150/hour
Billable Hours/Week: 20-30
Annual Revenue Y1: €50,000-120,000
```

## Business Type Selection Logic

### In Intake Module

```typescript
function determineBusinessType(answers: IntakeAnswers): BusinessType {
  // Check for regulated professions first
  if (answers.isFreiberufler) return 'Freiberufler';
  if (answers.requiresMeister) return 'Handwerk';

  // Check gastronomy
  if (answers.isFood) {
    if (answers.isMobile) return 'FoodTruck';
    if (answers.isEvent) return 'Catering';
    return 'Restaurant';
  }

  // Check product vs service
  if (answers.sellsProducts) {
    if (answers.hasPhysicalStore && answers.hasOnlineStore) return 'Hybrid-Handel';
    if (answers.hasPhysicalStore) return 'Einzelhandel';
    return 'E-Commerce';
  }

  // Service businesses
  if (answers.isDigital) {
    if (answers.isSubscription) return 'SaaS';
    return 'IT-Dienstleistung';
  }

  if (answers.isHealthcare) return 'Gesundheit';
  if (answers.isMobileService) return 'Mobile-Dienste';
  if (answers.hasTeam) return 'Agentur';

  return 'Beratung'; // Default for B2B services
}
```
